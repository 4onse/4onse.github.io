Documentation - istSOS communication library

Data transmission

The istSOS communication library enables the transmission of the data to the istSOS software. The data transmission can be perform using the GPRS 2G mobile connection of through the WiFi. The library supports boards with the SIM800 family module with a hardware or software wake-up function respectively implemented using the Sim800 or Drok classes. In addition the WiFi communication using the ESP8266 was implementing, but still not properly tested.

Data storage

The library provides a useful class (sdIstsos) to implement data strategy storaging using the OpenLog SD module. Using the functions developed, the 4onse weather station archives data into two different folder:

  • folder TMP where data to be sent are placed;
  • folder LOG where data sent are stored.
In fact, thanks to this library, the interaction with the OpenLog funcionalities is easyer and read, delete or create new file or folder is much less complex.
